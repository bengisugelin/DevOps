Let's see how we can store the variables and can retrieve them fro the bas shell.

store a variable:
```
[root@scriptbox ~]# SKILL="DevOps"
```
to retrieve it, just put a $ signt before the name of the variable:
```
[root@scriptbox ~]#  echo $SKILL
```
